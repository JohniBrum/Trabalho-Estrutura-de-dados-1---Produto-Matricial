# Trabalho-Estrutura-de-dados-1---Produto-Matricial
Programa em C que realiza o produto de duas matrizes A(m×n) e B(n×r) utilizando recursão, ponteiros e alocação dinâmica de memória. O usuário define as dimensões das matrizes em tempo de execução e escolhe entre inserir os valores manualmente ou gerá-los de forma pseudoaleatória, exibindo ao final a matriz resultante.
